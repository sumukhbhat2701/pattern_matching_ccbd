#### Installation:
<br/>
$ sudo apt install iverilog gtkwave
<br/>

#### Execution: 
<br/>
(Change the target filename in tb_regex.v if necessary) 
<br/>
● $ make 
<br/>
● Execute the binary files to simulate and record the waveforms. Eg:
<br/>
○ $ ./regex
<br/>
● Open “regex.vcd” file using GTKWave 
<br/>
○ $ gtkwave regex.vcd
<br/>
